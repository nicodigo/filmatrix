Principios

Cinematográfico

El contenido visual lidera. El poster es el protagonista; la UI es el marco que desaparece.

Denso y legible

Alta densidad de información sin sacrificar jerarquía. Cada nivel de texto tiene un rol único.

Dark-first

El fondo oscuro no es un tema, es la base del sistema. Todo contraste se mide sobre él.

Acento con propósito

El púrpura aparece solo en elementos accionables o de énfasis real. No como decoración.
